# Oracle 19c Single Instance 설치 가이드

Oracle Linux 7.9 환경에서 Oracle 19c Single Instance를 파일시스템 기반으로 설치하는 가이드입니다.

---

### 환경 구성

| 항목 | 내용 |
|------|------|
| OS | Oracle Linux 7.7 |
| DB 버전 | Oracle Database 19c |
| 스토리지 | 파일시스템 (Non-ASM) |
| 가상화 | VirtualBox 7.0.18 |
| 메모리 | 4GB |

---

### 서버 구성

| VM 이름 | 호스트 이름 | 메모리 | 어댑터 | 구성 방법 |
|---------|-----------|--------|--------|---------|
| ora19c_single | ora19c | 4GB | 어댑터에 브리지 | 리눅스 설치 |

---

### 스토리지

| 파일 이름 | 용량 | 타입 | 용도 |
|----------|------|------|------|
| ora19c_single | 50GB | Dynamic | 서버의 시스템 영역 |

### 주요 특징

- ASM 없이 파일시스템 기반의 간결한 Single 구성
- OGG 실습을 위한 orcl / clonedb 두 개 DB 생성
- listener.ora HOST 수정 및 방화벽 설정
- 실제 설치 중 발생한 오류 및 해결 방법 포함
