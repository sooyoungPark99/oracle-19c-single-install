## 3. MobaXterm SSH 접속

설치 완료 후 MobaXterm으로 SSH 접속한다.

- Host: 설정한 IP
- Username: `oracle`
- Port: `22`

```bash
# 접속 확인
hostname
# ora19c

ip addr
# 설정한 IP 확인
```
